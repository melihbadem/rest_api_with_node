## Rest API with Node
A basic REST API project written in Node.js/Express.

### for install
```console
git clone https://github.com/melihbadem/rest_api_with_node.git
````
install packages after installing

```console
npm install
```

run the application
```console
node app.js
```

and see it
```console
localhost:3000
```
